== Description ==

Плагин Vavilon Preloader необходим для возможности скрытия ужаса подгрузки темы на сайтах системы Wordpress.

== Installation ==
1.Скаченный плагин установить через админку или в ручном режиме поместить в папу plugins
2.В административной части сайта в пункте плагины Вы увидите новый плагин "Vavilon Preloader".
3.Активируете его.
3.Пользуйтесь на здоровье.

=== After activation ===
 После активации не нужно делать никаких манипуляций.
